# Calzado Santi App

## Descripción
Aplicación creada en React Native para la venta y distribución de zapatos, 
permitiendo ver catálogos, localización de tiendas, generar carrito de compras, 
visualizar tamaños de calzado, etc.

## Tecnologías
* React Native
* Typescript
* Axios
* React Query

## Arrancar proyecto 
1. npm i 
2. npm start
